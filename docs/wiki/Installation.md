# Installation

## Web

Visitez https://medlabib.github.io/EMtools/

## Android

Téléchargez l'APK depuis [Releases](https://github.com/medlabib/EMtools/releases)

## Build depuis les sources

```bash
git clone https://github.com/medlabib/EMtools.git
cd EMtools
flutter pub get
flutter run
```

## Prérequis

- Flutter 3.29+
- Dart 3.9+
