# EMtools

Outils numeriques pour la medecine d'urgence

## Fonctionnalites

- Evaluation ABCDE - Rapports medicaux structures
- Drogues Vasoactives - Calcul des doses
- Sedation - Protocoles de sedation-analgesie
- Gazometrie - Interpretation des gaz du sang
- Metabolique - Correction des troubles hydro-electrolytiques
- Antibiotiques - Guide de prescription
- Calculateurs - Scores cliniques

## Installation

Web: https://medlabib.github.io/EMtools/

Android: https://github.com/medlabib/EMtools/releases

Build:
git clone https://github.com/medlabib/EMtools.git
cd EMtools
flutter pub get
flutter run
